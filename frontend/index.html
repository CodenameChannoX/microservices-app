Architecture Overview

This application uses a microservices architecture with:

Frontend
Vue.js (Port 8080)
API Gateway
Node.js (Port 3000)
User Service
PostgreSQL (Port 3001)
Product Service
MongoDB (Port 3002)

Quick Start

  1. Access the API: http://localhost:3000
  2. Test User Service: http://localhost:3001/health
  3. Test Product Service: http://localhost:3002/health
  4. Seed sample products: POST http://localhost:3002/products/seed

API Endpoints

  • POST /api/auth/register
  • POST /api/auth/login
  • GET /api/auth/profile
  • GET /api/products
  • POST /api/products
  • GET /api/health
